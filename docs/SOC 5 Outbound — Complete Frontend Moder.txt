Project Overview

SOC 5 Outbound is an internal Truck Request Portal used by Shopee Sorting Facility Dispatch.

The system manages:

- Truck Requests
- Approval Workflow
- Truck Assignment
- Dock Assignment
- Dispatch Scheduling
- Realtime Status Updates
- User & Role Management
- Audit Trail
- Dashboard & Reporting

This is an internal enterprise application—not a public consumer-facing website.

The interface should prioritize:

- Speed
- Readability
- Operational efficiency
- Information density
- Accessibility
- Maintainability

- SCSS
- Sass variables
- Mixins
